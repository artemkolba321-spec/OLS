# OLS v0.1.0
OLS (Open Linux Shell) - This is a project aimed at improving the work with the terminal.

## installation
clone the code
```Bash
$ git clone git@github.com:artemkolba321-spec/OLS.git
$ cd OLS
```
next initialization
- if Bash
```Bash
$ bash ./init.sh
```
- if Zsh
```Bash
$ zsh ./init.sh
```
everything is ready
